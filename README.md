# 🚀 BTC Strategy Miner V3

비트코인 선물 거래를 위한 AI 기반 전략 발굴 및 백테스팅 플랫폼

## ✨ 주요 기능

- **🤖 자동 전략 발굴**: AI가 자동으로 수천 개의 전략을 테스트하고 최고의 전략을 발견
- **📊 고급 백테스팅**: DCA, Long/Short 스위칭, 손절/익절 등 실전 로직 구현
- **🏆 명예의 전당**: 발견된 최고 성과 전략들을 자동으로 순위화
- **🔴 실전/모의 매매**: Paper Trading 및 Binance Futures 실전 매매 지원
- **📈 상세 분석**: Sharpe Ratio, Sortino Ratio, Profit Factor 등 전문가급 지표

## 🛠️ 설치 방법

```bash
pip install -r requirements.txt
```

## 🚀 실행 방법

```bash
streamlit run app.py
```

## 📋 필수 요구사항

- Python 3.8 이상
- 최소 4GB RAM
- 인터넷 연결 (데이터 다운로드용)

## ⚠️ 주의사항

- 실전 매매 시 Binance Futures API 키가 필요합니다
- 투자 손실에 대한 책임은 사용자에게 있습니다
- 백테스팅 결과가 미래 수익을 보장하지 않습니다
